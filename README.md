# Breast Cancer Classification using ML & Deep Learning 🧬

## 📌 Description
This project focuses on classifying breast cancer tumors as **Benign (0)** or **Malignant (1)** using multiple machine learning and deep learning models.

---

## 🔍 Project Workflow
- Data Loading & Exploration
- Data Cleaning (handling duplicates & missing values)
- Encoding target variable
- Feature Selection using Correlation Analysis
- Data Visualization (heatmap, pairplots, distributions)
- Outlier Detection & Removal
- Data Splitting (Train/Test)
- Model Training & Evaluation

---

## 🤖 Models Used
- Support Vector Machine (SVM) with multiple kernels:
  - Linear
  - RBF
  - Polynomial
- Neural Network (Deep Learning using TensorFlow/Keras)

---

## 🛠️ Tools & Technologies
- Python
- Pandas & NumPy
- Matplotlib & Seaborn
- Scikit-learn
- TensorFlow / Keras

---

## 📊 Results
- SVM achieved high accuracy across different kernels.
- The **Linear Kernel** performed best among SVM models.
- Neural Network also achieved a strong performance.
- SVM slightly outperformed the Neural Network in accuracy.

---

## 📉 Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix
- ROC Curve & AUC

---

## 📈 Visualization
- Correlation Heatmap
- Pairplots for feature relationships
- Boxplots for outlier detection
- Confusion Matrices
- ROC Curve comparison (SVM vs Neural Network)

---

## 🧠 Key Insights
- Selected features (e.g., radius, perimeter, concavity) strongly influence classification.
- Feature scaling improved model performance.
- SVM proved highly effective for this dataset.
- Neural Networks provided competitive results but required more tuning.

---

## ⚖️ Model Comparison
- Both models performed well without overfitting.
- SVM showed slightly better accuracy and stability.
- Neural Network can be improved with hyperparameter tuning.
